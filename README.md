# Vehicle Tracking and Counting using AI

## Introduction
This project detects, tracks, and counts vehicles using Artificial Intelligence and Computer Vision techniques.

## Objective
The main objective of this project is to monitor traffic automatically using AI-based vehicle detection and tracking methods.

## Technologies Used
- Python
- OpenCV
- YOLOv8
- DeepSORT
- NumPy

## Features
- Vehicle Detection
- Vehicle Tracking
- Vehicle Counting
- Real-time Traffic Monitoring
- Multiple Vehicle Classification

## Project Workflow
1. Capture video from traffic camera
2. Detect vehicles using YOLO model
3. Track vehicles using DeepSORT
4. Count vehicles crossing a virtual line
5. Display traffic statistics

## Applications
- Smart Traffic Management
- Toll Booth Monitoring
- Parking Management
- Highway Traffic Analysis
- Smart City Systems

## Future Enhancements
- Number Plate Recognition
- Speed Detection
- Accident Detection
- Traffic Density Analysis

## Conclusion
This project helps automate vehicle monitoring and improves traffic management efficiency using Artificial Intelligence.
